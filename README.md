# READ ME FILE

HOW TO DOWNLOAD EasyEyes:
1. download this repository as a zip file ???
2. extract the zip file as a folder
3. in Google Chrome, search: chrome://extensions
4. on the top right of the screen, toggle on developer mode
5. on the top left of the screen, click the "Load unpacked" button
6. upload the folder you created
7. EasyEyes will appear under "All extensions"
8. pin this extention (HOW TO PIN IT?)
    - click the puzzle icon beside your browser's search bar
    - click on the pin icon beside the EasyEyes extension
9. run it on any website!
    - to test this extention, visit this website, and follow the proceeding steps: 
        https://docs.racket-lang.org/htdp-langs/beginner.html

HOW TO USE EasyEyes:
1. beside the search bar, click on the EasyEyes logo
2. toggle on the modifications you would like to make
